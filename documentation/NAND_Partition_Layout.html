mtd1 — UpdatableLBI (5 MB) READ-ONLY
Primary Linux Boot Image (4.01 MB). Contains the second-stage bootloader and Linux kernel in HP's proprietary LBI container format. The FSBL chain-loads the 2nd-stage bootloader from here, which then sets up the DTB, decompresses the kernel, and boots Linux.
